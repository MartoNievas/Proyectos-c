# Objetivo del Repo

El objetivo de este repo es mantener un seguimiento de mis proyectos en el lenguaje de programacion C.

## Proyectos:

### Function Plotter

- Utilizando SDL para poder imprimir por pantalla la funcion correspondiente y usando la libreria tinyexpr para parsear la funciones matematicas y evaluarlas en el dominio correspondiente.

### File Splitter

- Cuenta con la funcionalidad de splittear los archivos en n partes y luego con la funcion de juntar los archivos que se quieran en un solo usando las flags -s (splittear) y -j (join) para juntar los archivos.
